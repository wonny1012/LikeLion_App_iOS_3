## 조건문(while)

### 예제 7
PlusGame
랜덤으로 정수 2개를 뽑아 아래와 같이 출력
- 사용자는 두 수의 합을 입력
- 두 수의 합과 입력한 수가 일치하면 “Success”
- 두 수의 합과 입력한 수가 일치하지 않으면 “Fail”을 출력
```
=== Plus Game ===
12+17= 30
Success
```
```
=== Plus Game ===
12+17= 20
Fail
```
